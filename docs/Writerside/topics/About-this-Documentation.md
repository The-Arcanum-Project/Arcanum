# About this Documentation

This documentation is designed to help you get started with **Arcanum** and guide you through its core features.

Please note that it is still a work in progress. Some features may not yet be fully documented, and new functionality may be added over time.

If you encounter unclear explanations, missing details, or have suggestions for improvements, we encourage you to reach out to us with feedback.

A **FAQ section** will also be added in the future once we identify commonly asked questions and recurring issues.
