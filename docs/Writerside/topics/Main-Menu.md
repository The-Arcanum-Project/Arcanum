# Main Menu

The **Main Menu** is the default landing page when you launch Arcanum.
It includes multiple tabs as listed below and provides the functionality to open or create a project.

## Quick Access Bar

The **Quick Access Bar** at the top provides quick access to your projects:

* **New Project:** Opens the [Arcanum Tab](#arcanum) to create a new mod configuration.
* **Last Project:** Launches your most recently used project.

> The last project button will switch its behavior depending on the tab you are currently in.
> In the Arcanum Tab it will launch the current configuration as entered instead of the last one.
> {style="note"}

## Tabs

### Home

The **Home Tab** helps you stay informed and up to date.

You can:

* Check for new Arcanum versions
* View the latest news and feature spotlights
* Access quick links to our GitHub and Discord

### Configure {id="arcanum"}

This panel is the central menu for adding and managing mods in Arcanum.

At the bottom, you will find a list of your recent projects, allowing you to quickly open any mod if you are working on
multiple projects simultaneously.

<procedure title="Create a New Project" id="create-new-project">
    <step>
        Set the game file location in the top input field.
        <p>You can enter the path manually or click the "Folder" icon to browse.</p>
        <tip>
            The game file location is detected automatically if installed with <a href="https://store.steampowered.com">Steam</a>.
        </tip>
    </step>
    <step>
        Select your mod's directory in the lower input field.
    </step>
    <step>
        <b>(Optional)</b> Add base mods by clicking the green "Folder" icon.
        <p>Drag and drop the added mods to change their load order or remove them.</p>
    </step>
    <step>
        Click the "Current Config" button in the top bar to launch Arcanum.
    </step>
</procedure>
