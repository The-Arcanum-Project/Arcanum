# Getting Started

**Arcanum** is a **WPF** application built with **.NET**, and is therefore currently only supported on **Windows** systems.

There are two straightforward ways to [install Arcanum](Install.md) depending on your preferences.

To get to know **Arcanum** we would recommend, reading the [](Quick-Start-Guide.md) to get up to speed with everything.
Later you can dive into the more advanced [features](Features.md).
