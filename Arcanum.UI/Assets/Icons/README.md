# Sorting of the icon files

- 16x16 -> Icons for menu and context menu icons
- 20x20 -> Icons for buttons
- 30x30 -> Icons for larger buttons and 'labels'
- 40x40 -> Unique larger icons, e.g. Window icons
- 64x64 -> Icons for larger buttons and labels, e.g. Map icons
- Tiny -> Icons smaller than 16x16, e.g. for tooltips
- Huge -> Icons larger than 64x64, e.g. for splash screens or large buttons
- Images -> Logos of discord, Patreon, git, and other used images
- Logo -> Arcanum logos
